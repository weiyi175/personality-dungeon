### 1.4 核心物理結論（已鎖定）

1. **k50(N) 非單調**：含 N=200–300 systematic dip/crossover，視為物理性 finite-size crossover，而非噪音。
2. **k∞ 為 model-dependent 工作點**：BMA 95% CI = [0.818, 0.956]；寬度主要源自模型不確定性（ΔBIC < 2 間多模型競爭）。
3. **β 為 conditional estimate**：固定 k∞≈0.9543 + xgrid 評分（transition region P∈[0.1,0.9]）→ β≈0.52，basin 寬（Δβ≈±0.04 內 score 差異 < 1%）。
4. **收斂狀態**：iter8 後 delta 驗證 ≈ 0（k50 per-N 無顯著改變），refinement 已收斂。

#### 3.3 論文 Scaling 段落模板（可直接修改使用）

```
We characterize the finite-size scaling of the critical diffusion threshold k50(N)
across system sizes N ∈ {50, 100, 200, 300, 500, 1000}, using S=120 independent
seeds per (k, N) point (R=4000 rounds, burn-in fraction 0.30, tail window=1000).

The estimated k50(N) exhibits non-monotone size dependence, with a pronounced dip
at N≈200–300 (Fig. 1), consistent with a finite-size crossover rather than
statistical noise; Bayesian credible intervals (Laplace approximation) confirm this
structure is not sampling artefact.

Model comparison over four scaling forms [A: k∞+c/N, B: k∞+c/N^β,
C: k∞+c/N+d/N², D: piecewise two-regime] via BIC yields competitive fits
(ΔBIC < 2 across all models; Fig. 2), indicating that k∞ is best reported as
a model-averaged estimate. Bayesian model averaging (BIC weights; 10,000 draws)
gives k∞ = 0.94 (median), 95% CI [0.82, 0.96] (Fig. 6), where the posterior
width reflects model uncertainty rather than sampling noise.

For data collapse, we adopt a two-step procedure: k∞ is first fixed at the BMA
median (≈0.9543 for the leading model), then β is chosen to minimise the
cross-N interpolation variance on a uniform x-grid restricted to the transition
region P(Level3) ∈ [0.1, 0.9] (xgrid scoring, n=81 grid points).
The optimal collapse exponent is β ≈ 0.52 (Fig. 3), with a flat basin covering
β ∈ [0.48, 0.58] (score increase < 2%), indicating that the collapse is robust
to small perturbations in β (Fig. 4 shows deterioration at β=1.2).
```

### 優先 2 ★★：理論解釋與 payoff 連結

**理由**：β ≈ 0.52–0.56 與 N=200–300 dip 是強物理訊號，需要對應理論敘事。

#### 3.4 建議的討論方向（選 1–2 個深入）

1. **β ≈ 0.52 的 universality class 討論**
   - 比較：KPZ（β≈1/3）、Ising 2D（β≈0.125）、MF percolation（β=1）、voter model（β→∞/log）
   - 此系統是否與 1D directed percolation（β≈0.277）或 long-range 耦合有關聯？
   - 若 β≈0.52 不對應任何標準 universality class，可能代表 crossover 效應（非真正臨界點的 effective exponent）。

2. **N=200–300 dip 機制**
   - Mean-field 分析：用 payoff matrix (a=0.4, b≈0.2425, η=0.55) 建一個 ODE，推算確定性臨界 k，與 k50(N→∞)≈0.94 是否吻合。
   - Dip 可能的物理解釋：有限族群中的 "resonance" 尺度（對應某特徵擴散長度），或 stage2/stage3 定義的 ambiguity 在特定 N 下放大。
   - 可驗證：對照組 lag=0 或 lag=2 的 k50(N) 曲線——若 dip 位置隨 lag 移動，則 lag 是關鍵參數。

3. **P(Level3) vs ρ⁻¹ 為 control parameter**
   - 目前 x-axis 是直接 k；也可嘗試 x = k − k∞（以 k∞ 為 BMA median）做 collapse，看 scaling 是否更整齊。

#### 3.5 簡單 mean-field 推導（若要實作）

```bash
# 數值確認：固定 a=0.4, b=0.2425407441, eta=0.55, lag=1
# 推導 deterministic k_crit（ODE fixed-point collapse condition）
./venv/bin/python - <<'PY'
import numpy as np

a, b, eta, lag = 0.4, 0.2425407441, 0.55, 1

# Mean-field payoff for cooperation under replicator dynamics
# Approximate: P(Level3 | k) ~ Phi((k - k_crit) / sigma_eff(N))
# k_crit from ODE: where cooperation fraction crosses 0.5

# For reference: k50(N→∞) empirical ≈ 0.94 (BMA median)
# Compare to a simple MF estimate:
k_mf_estimate = (a + b * eta) / (1 - eta + eta * b)
print(f'Simple MF k_crit estimate: {k_mf_estimate:.4f}')
print(f'Empirical BMA k_inf median: 0.941')
PY
```
