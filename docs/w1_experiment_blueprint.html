  • 固定節奏執行 A3，萃取核心指標。
  • 整理既有 B2/B3 失敗 block 的 seed 級資訊。

2.2 Out of Scope

  • 不做 B2/B3 參數延伸格點。
  • 不重啟 Event line smoke/gate 新實驗。

  2. 從 A3 summary 擷取核心欄位，寫入看板資料表。
  3. 生成 W1 的 A3 trend snapshot（最小版可用表格）。

6. 風險與停損

風險：資料整理工作膨脹成重新掃參。
  • 停損 1：W1 不新增任何新參數 sweep 命令。
  • 停損 2：若 catalog 欄位超過核心欄位兩倍，先縮欄位再擴資料量。
  • 停損 3：若 A3 fail，W1 其餘探索工作暫停，先修主線。
W1 不是「找新最佳參數」，而是「把回歸治理與失敗知識庫做成可累積資產」。
